# System - Monolith (C#)

## Instructions

1. Open up the 'monolith' folder in Visual Studio (make sure it is at the root of your project)

2. Ensure you have .NET 8:

```shell
dotnet --version
```

3. To build the project:

```shell
dotnet build
```

To run:

```shell
dotnet run --urls "http://localhost:8080"
```
